## 判断基準まとめ

### すぐ追加

- 書きたいことが**3 つ以上**溜まった
- 同じことを**3 回**調べた/書いた
- **忘れそう**だと感じた

### 様子見

- あったら便利かも？ → 一旦メモに書いて様子見
- 他のプロジェクトで使ってた → 本当に必要か考える

### 追加しない

- 更新する自信がない
- 既存ファイルで十分カバーできる
- GitHub の機能で代替できる（Issue、Wiki 等）
